Hackathon Organizer

A web application for hackathon organizers to plan and manage events. Centralizes planning information instead of using separate tools like Google Sheets, Docs, Discord, and Slack.

Team: Joseph Davis Chamdani, Kenneth Wu, Rowan Cooper, Winson Teh

Live Proposal: View on GitHub Pages

GitHub Repo: info340-wi26a/project-JosephDavisC

Features (2.5 Required)

  • Task Management (1.0): Create tasks, assign to team members, set priorities, mark complete, filter and reorganize
  • Team Management (1.0): Create organizing teams (logistics, marketing, sponsorship, judging), assign responsibilities
  • Milestone Tracking (0.5): Dashboard for milestones like registration deadlines, sponsor confirmations, schedule releases
