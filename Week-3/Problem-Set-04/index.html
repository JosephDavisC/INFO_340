Problems

  • Problem A Responsive Splash Page - Media queries and mobile-first design
  • Problem B Bootstrap Cards - Grid system and responsive components

Key Concepts

  • Viewport Meta Tag: <meta name="viewport" content="width=device-width, initial-scale=1">
  • Media Queries: @media (min-width: value) for breakpoint-based styling
  • Mobile-First: Default styles for mobile, enhanced for larger screens
  • Bootstrap Grid: 12-column responsive layout with col-* classes
  • Bootstrap Cards: card, card-body, card-title, card-text
  • Utility Classes: Spacing (p-*, m-*), colors (bg-*, text-*), display (d-flex)
