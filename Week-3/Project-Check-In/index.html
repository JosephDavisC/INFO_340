Project Check-In

Lab - January 21, 2026

Wireframes

We used Figma to design our mockups for our project.

Our wireframes include 4 main screens:

  • Homepage: "Plan Your Hackathon in One Place" landing page with Get Started button
  • Sign Up: Create Account form with name, email, password fields and Google sign-up option
  • Dashboard: Main interface with sidebar navigation (Dashboard, Tasks, Teams), Recent Tasks list, and Upcoming Milestones tracker
  • Login: Welcome Back login form with email/password and Google sign-in

Team Communication

For communication, we set up a Discord server.

Why This Project?

We chose this project idea because several of us have experience organizing hackathons and understand how chaotic it can get when planning info is scattered across different tools. We wanted to build something that solves a problem we've actually faced.

Progress

So far, everything is going well and everyone is contributing nicely!
