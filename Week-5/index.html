Key Concepts

  • DOM: Document Object Model - tree of element nodes from HTML
  • Selecting Elements: querySelector(), querySelectorAll()
  • Events: addEventListener(), event handlers, event.preventDefault()
  • Arrow Functions: Shorter syntax with =>, implicit return for expressions
  • Destructuring: Unpack arrays [a, b] or objects {x, y}
  • Spread Operator: ... expands elements into new arrays/objects
  • ES6 Modules: import/export for code organization
